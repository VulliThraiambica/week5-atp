# week 5 Assignments

This folder contains simple React components and HTML/CSS practice programs for learning frontend development basics.

---

# React Files

## App.jsx

Main React component of the application.  
Connects and displays all other components together.

### Concepts Used
- React Components
- JSX
- Component Rendering

---

## Navbar.jsx

Creates a navigation bar for the webpage.  
Displays menu items or links.

### Concepts Used
- Functional Components
- Navigation Design
- JSX

---

## Footer.jsx

Creates the footer section of the webpage.  
Displays footer text or information.

### Concepts Used
- React Components
- Page Layout

---

## Product.jsx

Displays product details using React.  
Shows product information like name or price.

### Concepts Used
- Props
- JSX
- Components

---

## User.jsx

Displays individual user information.  
Shows user details using props.

### Concepts Used
- Props
- Functional Components

---

## UserList.jsx

Displays multiple users using arrays and loops.  
Renders user data dynamically.

### Concepts Used
- Arrays
- map()
- Component Rendering
